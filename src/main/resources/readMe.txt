使用方式：

1.  配置config.properties
    选择数据源、生成路径、要生成的模块

2.  如只需生成默认风格（fpp风格）的代码，则直接执行Test类main方法即可
    如需生成自定义风格的代码，则修改/resources/template下的各种模板，再运行test即可
